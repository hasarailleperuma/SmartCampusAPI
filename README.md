# SmartCampusAPI

This is a Java-based RESTful API project developed using Maven and JAX-RS for Client-Server Architecture coursework.

## Features
- REST API development using JAX-RS
- Maven project structure
- Modular package design
- HTTP request handling
- Git version control integration

## Technologies Used
- Java
- Maven
- JAX-RS
- IntelliJ IDEA
- Git & GitHub

## Purpose
This project demonstrates backend development concepts and RESTful web service implementation for academic learning.
